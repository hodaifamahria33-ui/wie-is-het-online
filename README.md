# Wie Is Het? Online

Alleen het **beginscherm** — titel, gele LET'S BEGIN-knop, donkere achtergrond met wazige kaartjes.

## Bekijken in Unity

1. Installeer [Unity Hub](https://unity.com/download) + **Unity 2022.3 LTS**
2. Unity Hub → **Open** → `C:\Users\hodai\WieIsHetOnline`
3. Wacht tot Unity klaar is met importeren
4. Druk **Play** ▶

Het beginscherm verschijnt automatisch. Geen extra setup nodig.

**Optioneel (eerste keer):** menu **Wie Is Het → Setup Start Screen Scene** en daarna Play.
